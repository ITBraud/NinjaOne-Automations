# NinjaOne-Automations
A centralized repository for developing, versioning, and sharing **NinjaOne RMM** scripts, automations, and application deployment packages.

Built for MSPs and internal IT teams to standardize device setups, software deployments, and recurring tasks across Windows endpoints.

## Purpose

This repo is the single source of truth for reusable NinjaOne automations, especially complex line-of-business deployments like Dynamics GP 2018 TimeClock.
